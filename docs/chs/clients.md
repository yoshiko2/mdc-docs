# MacOS
### 允许任何来源程序运行
* 左上角 - `系统偏好设置` - `安全性与隐私` - `安全性` - 勾选`任何来源`
* 如果没有，请在`终端`运行 `sudo spctl --master-disable` 密码为用户登陆密码
* 在`终端`运行 `sudo spctl --master-disable` 密码为用户登陆密码
* MacOS ARM也能运行amd64

由于目前开发者贫穷买不起Mac mini编译程序，转译运行的某些代码无法识别系统信息，M系列暂时无法记住密码，请在用户面板踢出设备后登录
